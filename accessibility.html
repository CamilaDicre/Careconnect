Accessibility Statement

Committed to making Careconnect accessible for everyone

Our Commitment to Accessibility

Careconnect is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards to make our platform inclusive for all users, especially seniors and individuals with diverse abilities.

Accessibility Features

Our platform includes the following accessibility features designed to enhance usability for all users:

Visual Accessibility

  • High contrast color schemes and large, readable fonts
  • Clear typography with adequate spacing and line height
  • Consistent visual hierarchy and navigation
  • Alternative text for images and icons

Keyboard Navigation

  • Full keyboard accessibility for all interactive elements
  • Visible focus indicators for keyboard users
  • Logical tab order throughout the platform
  • Keyboard shortcuts for common actions

Screen Reader Support

  • Compatible with popular screen readers (JAWS, NVDA, VoiceOver)
  • Semantic HTML structure for proper navigation
  • Descriptive labels and ARIA attributes
  • Clear heading structure and landmarks

Motor and Mobility Support

  • Large touch targets for mobile devices
  • Adjustable timing for form submissions
  • Voice control compatibility
  • Gesture alternatives for complex interactions

Cognitive and Learning Support

  • Simple, clear language and instructions
  • Consistent interface design and navigation
  • Error prevention and clear error messages
  • Multiple ways to complete tasks

Compliance Standards

We strive to conform to the following accessibility standards:

  • WCAG 2.1 AA: Web Content Accessibility Guidelines
  • Section 508: Rehabilitation Act of 1973
  • ADA Title III: Americans with Disabilities Act
  • EN 301 549: European accessibility standard

Ongoing Improvements

We are continuously working to improve accessibility by:

  • Regular accessibility audits and testing
  • User feedback and testing with people with disabilities
  • Training our development team on accessibility best practices
  • Monitoring and implementing new accessibility technologies
  • Regular updates to meet evolving accessibility standards

Testing and Evaluation

Our accessibility testing includes:

  • Automated accessibility testing tools
  • Manual testing with assistive technologies
  • User testing with individuals with disabilities
  • Expert accessibility reviews
  • Regular compliance assessments

Assistive Technology Compatibility

Our platform is designed to work with the following assistive technologies:

  • Screen Readers: JAWS, NVDA, VoiceOver, TalkBack
  • Screen Magnifiers: ZoomText, Magnifier
  • Voice Recognition: Dragon NaturallySpeaking, Voice Control
  • Switch Devices: Various switch control systems
  • Braille Displays: Compatible with screen reader output

Browser and Device Support

We support accessibility features across multiple browsers and devices:

  • Chrome, Firefox, Safari, and Edge browsers
  • Mobile devices (iOS and Android)
  • Tablets and desktop computers
  • High-DPI displays and various screen sizes
